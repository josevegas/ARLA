# Agentes Personalizados para ARLA

Este archivo define agentes personalizados para el desarrollo y mantenimiento de la aplicación ARLA.

## Agente Backend Developer
- **Propósito**: Manejar tareas relacionadas con el backend, como API, base de datos y lógica del servidor.
- **Habilidades**: Node.js, Express, TypeScript, Prisma, PostgreSQL.
- **Activación**: Cuando se mencione "backend" o archivos en `/backend/`.

## Agente Frontend Developer
- **Propósito**: Gestionar el desarrollo del frontend, UI/UX y componentes React.
- **Habilidades**: React, Vite, TypeScript, Tailwind CSS, Neumorfismo.
- **Activación**: Cuando se mencione "frontend" o archivos en `/frontend/`.

## Agente Tester
- **Propósito**: Ejecutar y mantener tests, asegurar calidad del código.
- **Habilidades**: Jest, Supertest, testing de APIs.
- **Activación**: Archivos en `/TEST/` o comandos de testing.

## Agente DevOps
- **Propósito**: Manejar configuración de Docker, despliegue y CI/CD.
- **Habilidades**: Docker, docker-compose, scripts de build.
- **Activación**: `docker-compose.yml` o comandos de despliegue.
